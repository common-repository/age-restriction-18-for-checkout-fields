Se sua loja Woocommerce utiliza o plugin WooCommerce Extra Checkout Fields for Brazil e precisa limitar o cadastro de menores de 18 anos, esse é plugin perfeito para você.
O age restriction 18+ for Checkout fields impossibilita o cadastros de usuários menores de 18 anos. Em suas possibilidades de uso estão: loja de bebidas, sex shop, medicamentos ou qualquer outra aplicação que necessite limitar a idade ao cadastrar o usuário para compra.
Esse plugin é um complemento do plugin [WooCommerce Extra Checkout Fields for Brazil](https://wordpress.org/plugins/woocommerce-extra-checkout-fields-for-brazil/) e só funcionará com ele instalado.

== Description ==

Se sua loja Woocommerce utiliza o plugin WooCommerce Extra Checkout Fields for Brazil e precisa limitar o cadastro de menores de 18 anos, esse é plugin perfeito para você.
O age restriction 18+ for Checkout fields impossibilita o cadastros de usuários menores de 18 anos. Em suas possibilidades de uso estão: loja de bebidas, sex shop, medicamentos ou qualquer outra aplicação que necessite limitar a idade ao cadastrar o usuário para compra.
Esse plugin é um complemento do plugin [WooCommerce Extra Checkout Fields for Brazil](https://wordpress.org/plugins/woocommerce-extra-checkout-fields-for-brazil/) e só funcionará com ele instalado.

= Compatibilidade =

Compatível com a versão 5.2 Wordpress


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/age-restriction-18-for-checkout-fields` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Woocommerce->Age +18 restriction screen to configure the plugin
